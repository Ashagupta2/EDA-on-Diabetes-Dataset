## Project Description
This project performs Exploratory Data Analysis (EDA) on the Diabetes dataset. The dataset contains medical attributes such as pregnancies, glucose level, blood pressure, skin thickness, insulin, BMI, diabetes pedigree function, and age. The aim of this analysis is to understand data patterns, find missing or unusual values, detect outliers, and identify important factors related to diabetes risk.

## Dataset Attributes
| Attribute | Description |
|---|---|
| Pregnancies | Number of times the patient has been pregnant |
| Glucose | Blood glucose level of the patient |
| BloodPressure | Blood pressure value |
| SkinThickness | Skin fold thickness measurement |
| Insulin | Insulin level in the blood |
| BMI | Body Mass Index, calculated from weight and height |
| DiabetesPedigreeFunction | Indicates genetic/family history influence of diabetes |
| Age | Age of the patient |

## Objectives
- Understand the structure of the dataset
- Check missing values and duplicate records
- Analyze feature distributions
- Detect outliers using boxplots
- Study relationships between attributes
- Identify important factors related to diabetes

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / Kaggle Notebook

## EDA Steps Performed
- Loaded and explored the dataset
- Checked dataset shape, columns, and data types
- Analyzed summary statistics
- Checked missing and duplicate values
- Visualized distributions using histograms
- Detected outliers using boxplots
- Studied correlations between attributes using heatmap

## Key Insights
- Glucose is one of the most important attributes related to diabetes risk.
- BMI and Age also show important patterns in the dataset.
- Insulin and SkinThickness may contain outliers.
- Some medical values may need cleaning before machine learning model building.

## Conclusion
This EDA helps in understanding the Diabetes dataset and identifying important medical attributes that may influence diabetes risk. The analysis can be used as a base for future machine learning model development.
